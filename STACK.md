# Стек проекта

## IDE и инструменты
- VS Code + Claude (расширение)

## Модели
- `claude-haiku-4-5-20251001` — персонализация (быстро + дёшево)
- Стоимость генерации на 55 компаний: **~$0.04 (4 цента)**

## Технологии
- Node.js 20, ES Modules
- cheerio — парсинг HTML
- node-fetch — HTTP запросы
- @anthropic-ai/sdk — Claude API
- csv-writer — экспорт результатов
- dns/promises — MX валидация email (встроен в Node.js)

## Источники данных
