# 05 — Критерии оценки тестового задания

## Оригинальное ТЗ

**Продукт:** Polza Agency — агентство по запуску email-аутрича для B2B-компаний  
**Целевой клиент:** B2B-компания, которая хочет получать входящие заявки через холодные рассылки

---

## Задача 1: База компаний

**Требования:**
- Минимум 50 компаний
- B2B, Россия, есть отдел продаж или активно ищут клиентов
- Для каждой: название, сайт, имя, email
- Результат: Google Таблица

**Критерий "высокий вес":** реальные, валидные email

**Чеклист выполнения:**
- [ ] 50+ строк в таблице
- [ ] Каждая строка: company_name + website + email + contact_name
- [ ] Email прошли MX-валидацию
- [ ] Источник email указан (сайт / угадан / hh)
- [ ] Нет дублей компаний
- [ ] Все компании — реальные, живые B2B

---

## Задача 2: Персонализация

**Требования:**
- Короткий персональный факт о компании или ЛПР (1-2 предложения)
- Источник: сайт, LinkedIn, новости — не выдумывать
- Реализовать через скрипт: на входе список компаний, на выходе колонка «Персонализация»
- Приложить скрипт (GitHub / файл)

**Критерий "высокий вес":** персонализация осмысленная, не шаблонная

**Чеклист выполнения:**
- [ ] Колонка `personalization` заполнена для каждой строки
- [ ] Тексты разные (не один шаблон для всех)
- [ ] Есть привязка к конкретному факту / сигналу компании
- [ ] Скрипт выложен на GitHub
- [ ] README описывает как запустить
- [ ] Скрипт реально работает (не просто заглушка)

---

## Задача 3: Цепочка писем

**Требования:**
- 3 письма от имени Polza Agency
- Письмо 1: холодное знакомство с крючком
- Письмо 2: follow-up через 3 дня, другой угол / кейс
- Письмо 3: финальный follow-up через 5 дней, мягкий выход
- Каждое: тема + тело, до 120 слов
- В письме 1 — переменная {{персонализация}}
- Оформить отдельным листом в таблице

**Критерий "высокий вес":** звучит по-человечески, есть логика прогрева

**Чеклист выполнения:**
- [ ] 3 письма в отдельном листе Google Sheets
- [ ] Каждое: тема + тело
- [ ] Письмо 1 содержит `{{персонализация}}`
- [ ] Письмо 2 — новый угол, не повтор письма 1
- [ ] Письмо 3 — мягкий выход, без давления
- [ ] Каждое письмо ≤ 120 слов
- [ ] Тексты звучат как от живого человека
