# 03 — Персонализация через Claude API

## Философия

Старый подход (плохо):
> "Компания занимается разработкой ПО в Москве"

Новый подход (хорошо):
> "Вижу, вы ищете руководителя отдела продаж — значит масштабируете продажи. Мы помогаем таким компаниям получать поток встреч через точечный аутрич, пока строится команда."

**Правило:** персонализация строится на **сигнале** (конкретном событии), а не на описании отрасли.

---

## Типы сигналов

| Сигнал | Источник | Что пишем в письме |
|---|---|---|
| Ищут менеджера по продажам | HH вакансия | "Масштабируете продажи — аутрич ускорит результат" |
| Ищут руководителя продаж | HH вакансия | "Строите отдел продаж — дадим поток лидов сразу" |
| Запускают новый продукт | Сайт / новости | "Выходите на рынок — холодные письма дадут первых клиентов" |
| Есть свежие кейсы | Сайт | "Есть результаты — покажем их нужным ЛПР через аутрич" |
| Рейтинг Ruward | Ruward | "Топ рейтинга — время масштабировать поток лидов" |

## Claude API — конфигурация

```javascript
import Anthropic from '@anthropic-ai/sdk';

const client = new Anthropic();
// ANTHROPIC_API_KEY берётся из process.env автоматически

const response = await client.messages.create({
  model: 'claude-haiku-4-5-20251001', // быстро + дёшево
  max_tokens: 200,
  messages: [{
    role: 'user',
    content: buildPrompt(company)
  }]
});

return response.content[0].text.trim();
```

## Промпт (основной)

```javascript
function buildPrompt({ company_name, signal, site_text }) {
  return `Компания: ${company_name}
Сигнал: ${signal}
Текст с сайта: ${site_text.slice(0, 1500)}

Напиши 1-2 предложения персонализации для холодного B2B письма от лица аутрич-агентства.

Требования:
- Используй конкретный факт о компании ИЛИ сигнал из вакансии
- Объясни, почему именно им нужен email-аутрич прямо сейчас
- Запрещено: "ведущая компания", "динамично развивается", "лидер рынка"
- Запрещено: общие слова без привязки к конкретной ситуации
- Только конкретика и логика "сигнал → потребность → решение"

Формат: просто текст, 1-2 предложения, без кавычек и markdown.`;
}
```

## Примеры хорошей персонализации

**Сигнал:** "Ищут Head of Sales"
```
Вижу, вы открыли позицию Head of Sales — значит впереди масштабирование.
Пока строится команда, аутрич даст первый поток встреч уже в первый месяц.
```

**Сигнал:** "Ищут менеджера по продажам + свежие кейсы на сайте"
```
У вас есть кейсы с измеримыми результатами — это сильный оффер для холодных писем.
Через точечный аутрич эти результаты увидят именно те ЛПР, кому они релевантны.
```

**Сигнал:** "Ищут коммерческого директора"
```
Открытая позиция коммерческого директора говорит о росте — поможем не терять время
на поиск лидов вручную и дадим предсказуемый поток B2B-заявок уже сейчас.
```

---

## Стоимость и оптимизация

- Модель: `claude-haiku-4-5-20251001` (~$0.00025 за 1K input tokens)
- На 60 компаний: ~$0.05-0.10 (несущественно)
- Если нужно сэкономить: обрезай site_text до 800 символов

## Fallback логика

```javascript
async function getPersonalization(company) {
  try {
    return await callClaude(company);
  } catch (err) {
    console.log(`⚠️  Claude error: ${err.message}, используем сигнал`);
    return company.signal; // fallback = просто сигнал с HH
  }
}
```
